Convention I have followed for this project

Public headers are going to be in include/${PROJECT_NAME}/ Private headers are going to be in src/inc/ All implementations are going to be in src/impl

The directory and file structure src/impl will represent the logical components of the projects, include/project and src/inc directories should try to mimic this structure.

├── src

│ │ └── main.cpp

include/cmake-init/cmake_init_lib.h provides the interface and src/impl/cmake_init_lib.cc is the implementation.

All private headers and implementations are requried for include/cmake-init/cmake_init_lib.h and src/impl/cmake_init_lib.cc are arranged in src/inc/cmake_init_lib/ and src/impl/cmake_init_lib/

Note: cmake_init_lib.h is a public header and any header file from src/inc/ must not be included in it
